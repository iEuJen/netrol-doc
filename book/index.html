# 介绍

Netrol 是一个用于浏览器端的 AJAX 请求库。 它采用集中存储的方式，管理项目的服务端接口。

# 特性

  • 集中式管理接口
  • 支持 Promise
  • 请求和响应的拦截，并支持对单一接口进行响应拦截
  • 取消请求
  • 针对状态码进行捕获

# 什么时候使用 Netrol

建议在支持 ES6模块 的单页面应用中使用，因为服务端渲染的多页面应用，一般情况下单个页面需要用到的接口数量并不是很多，这时候使用 Netrol 就会显得过于繁重。
